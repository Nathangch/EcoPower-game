# EcoPower ⚡ - Simulação de Redes Elétricas

EcoPower é uma plataforma educacional e sandbox de alta fidelidade para simulação de redes elétricas, focada em conceitos de engenharia e sustentabilidade.

## 🎒 Modos de Jogo
- **Modo Educativo:** 25 níveis didáticos que ensinam princípios de redes, balanceamento de carga e redundância.
- **Modo Sustentável:** Simulação avançada com fontes renováveis (Solar, Eólica, Hidro), ciclos de dia/noite e métricas ambientais.
- **Modo Criativo:** Ferramenta sandbox para construir cidades complexas e redes de distribuição do zero.

## 🚀 Como Jogar
### 🌐 Navegador (Recomendado)
Jogue instantaneamente através do link do GitHub Pages:
[**EcoPower Web Edition**](https://nathangch.github.io/EcoPower-game/)
*(Compatível com Chrome, Edge e Firefox)*

### 🖥️ Desktop (Executável Windows)
Se você quer a versão desktop sem precisar instalar o Python:
1. Baixe o arquivo `EcoPower.exe` na raiz deste repositório.
2. Execute diretamente no seu Windows.

### 🐍 Desktop (Script Python)
Se você tem o Python instalado e quer rodar o código fonte:
```bash
python game.py
```

---
## 🛠️ Funcionalidades Técnicas
- **Simulação Dinâmica:** Algoritmo BFS que distribui energia MW por MW para balanceamento de carga real.
- **Sistema Climático:** Ciclos de dia/noite e intensidade variável de vento/água que afetam a geração em tempo real.
- **Infraestrutura:** Múltiplos tipos de cabos, transformadores e baterias com lógica de carga/descarga.
- **Ferramentas:** Medidores de carga, detectores de falhas e ferramentas de manutenção.

---
## 🏆 Objetivos Pedagógicos
O jogo foi desenhado para ensinar:
1. **Lei das Malhas e Nós:** Como a energia se divide em caminhos paralelos.
2. **Resiliência N-1:** Importância de rotas alternativas para evitar apagões.
3. **Matriz Energética:** O impacto das fontes renováveis vs. fósseis na sustentabilidade.
